# Мониторинг беговой дорожки Citysports LS6

Подключается к беговой дорожке Citysports LS6 по Bluetooth (BLE) и в реальном времени
выводит скорость, дистанцию, время тренировки, оценку сожженных калорий и набор высоты.

## Требования

- Python 3
- [bleak](https://github.com/hbldh/bleak) -- библиотека для работы с BLE
- Включённый Bluetooth

## Установка и запуск

```bash
python3 -m venv .venv           # создать виртуальное окружение
source .venv/bin/activate       # активировать
pip install bleak               # установить зависимости
python3 treadmill_monitor.py    # запустить
```
По окончанию работы выйти из venv:
```bash
deactivate
```

## Использование

1. Скрипт запросит ваш **вес** (в кг, по умолчанию 92).
2. Затем -- **угол наклона** беговой дорожки (в градусах, по умолчанию ~2.39).
3. Начнется сканирование BLE-устройств.
4. Если найдено **только одно** устройство -- подключение произойдет автоматически.
   Если найдено несколько -- будет предложено выбрать из списка по номеру.
5. Начнется потоковый вывод данных. Для остановки нажмите `Ctrl+C`.

## Отображение данных

```
  Speed:   2.0 km/h  |  Distance:   1.65 km  |  Time: 00:51:29  |  Calories:   118 kcal  |  Elevation:    69 m
```

- **Speed** -- текущая скорость (км/ч)
- **Distance** -- пройденная дистанция (км)
- **Time** -- время тренировки с момента запуска дорожки (ЧЧ:ММ:СС)
- **Calories** -- оценка сожженных калорий (ккал) без учёта набора высоты
- **Elevation** -- набор высоты, рассчитанный как `distance * sin(наклон)`

## Как считаются калории

Текущая формула использует MET-значения для ходьбы по **ровной поверхности**:

| Скорость     | MET |
|--------------|-----|
| < 3.2 км/ч   | 2.5 |
| 3.2-4.0 км/ч | 3.0 |
| 4.0-5.6 км/ч | 3.5 |
| > 5.6 км/ч   | 4.0 |

Калории: `(MET - 1) * вес(кг) * время(ч)`
